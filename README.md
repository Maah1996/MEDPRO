# MEDPRO®

**Sistema Experto para determinar el nivel de documentación de una actividad.**

MEDPRO decide, de forma objetiva, repetible y auditable, qué nivel de documentación
requiere una actividad: *Sin documentación → Buena práctica → Checklist → Cartilla →
Instructivo → Procedimiento → Proceso*.

Basado en la metodología MEDPRO (Tomos I–X) y en las normas
**ISO 9000 · 9001 · 31000 · 45001 · 14001**.

## 🔗 Aplicación en línea
👉 https://maah1996.github.io/MEDPRO/

## Características
- Sistema experto con **motor de decisión desacoplado**: 6 criterios gatillantes + 15 criterios ponderados.
- El usuario nunca ve puntajes internos: recibe una **recomendación justificada**, los 3 factores más influyentes y un semáforo de criticidad.
- 10 pantallas: Inicio, Metodología, Información general, Gatillantes, Multicriterio, Resultado, Historial, Dashboard, Administración y Ayuda.
- **Configuración editable sin tocar código** (pesos, umbrales, colores, criterios, logo).
- Exportación a **PDF, Word y Excel**.
- Almacenamiento en **Firebase Firestore** con respaldo automático en el navegador (localStorage).
- Responsive (PC, notebook, tablet, móvil), tema claro/oscuro, tipografía Times New Roman con control de tamaño.

## Tecnología
HTML5 · CSS3 · JavaScript ES6 · Bootstrap 5 · Chart.js · jsPDF · SheetJS · Firebase (Firestore).
Aplicación de archivo único (`index.html`), sin necesidad de servidor.
